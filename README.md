## 安装与使用

1. 克隆仓库：`git clone <repo-url>`
2. 将对应 skill 目录复制到：
   `~/.claude/plugins/cache/{publisher}/{plugin}/{version}/skills/`
3. 在 `~/.claude/settings.json` 中启用对应插件
