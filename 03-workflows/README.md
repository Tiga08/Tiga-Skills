# 工作流

存放多步骤工作流定义，描述跨工具或跨 Skill 的完整操作流程。

## 用途

- 定义由多个步骤组成的复杂操作流程
- 记录步骤间的依赖关系和数据传递
- 提供可重复执行的标准化流程

## 文件规范

- 文件名：kebab-case，如 `deploy-and-verify.md`
- 编码：UTF-8
- 格式：Markdown
- 每个文件定义一个完整工作流

## 文件结构

```markdown
# 工作流标题

> 一句话说明用途

## 触发条件

（何时使用此工作流）

## 步骤

1. 步骤一
2. 步骤二
3. ...

## 预期结果

（完成后的状态）
```

## 登记

添加新工作流后，请在 [00-skill-index](../00-skill-index/README.md) 中登记。
